# ARRAY[]
EL arreglo es unidimensional 
es "similar" a una lista.
Ejemplo:
``` 
frutas[2] = ["pera", "manzana", "naranja"]
``` 
El indice indica la posicion de los elementos
y lo podemos buscar
``` 
frutas[1] 
``` 
# MATRIZ[][]
La matriz es bidimensionales
renglones y columnas

### Ejemplo: 
``` 

       0  1
matriz[1][1] = [1, 3]
               [2, 4]
``` 

el primer parametro es renglones y el segundo son columnas

para recorrer un matriz necesitamos un for anidados

## Ambos empiezan en la posicion 0
